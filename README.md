An end-to-end business intelligence project featuring a "What-If" revenue forecast engine for BMW Group, built in both Tableau and Power BI.
# BMW Global Sales & Revenue Forecast Dashboard

## 📊 Project Overview
This project analyzes BMW's global sales performance across different fuel types and regions. It features a **Revenue Sensitivity Engine** that allows executives to model the impact of price adjustments on total revenue.

### 🔗 Live Interactive Dashboard
* [View Interactive Tableau Dashboard](https://public.tableau.com/app/profile/mkhulu.mokutu/viz/BMWGlobalRevenueExecutiveDashboard/Dashboard1?publish=yes)

## 🛠 Tech Stack
* **BI Tools:** Tableau Public, Power BI Desktop
* **Data Processing:** Python/Pandas (Data generation & cleaning)
* **Theme:** BMW Corporate "Dark Mode"

## 💡 Key Insights (from BMW_sales_master.csv)
* **Revenue Leader:** North America ($74.5B) is the top-performing territory.
* **Electrification:** Electric vehicles command the highest average price premium at ~$77.9k.
* **What-If Logic:** A 5% price shift in high-volume regions (Europe/NA) projects a revenue lift of ~$4.1B.
